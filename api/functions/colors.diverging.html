Function diverging

  • diverging(scheme, colorspace?): Collection

  • A wrapper function for ColorBrewer's map of diverging color schemes.

    Parameters

    • scheme: DivergingScheme

      The name of the scheme.

    • colorspace: Colorspaces = undefined

      The color to retrieve the scheme in.

    Returns Collection

    The collection of colors from the specified scheme.

    Example

    import { diverging } from 'huetiful-js'



    console.log(diverging("Spectral"))
    //[
     '#7fc97f', '#beaed4',
     '#fdc086', '#ffff99',
     '#386cb0', '#f0027f',
     '#bf5b17', '#666666'
    ]
